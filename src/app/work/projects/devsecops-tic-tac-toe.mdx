### 🎮 Project Overview

Developed a secure Tic-Tac-Toe app using React, TypeScript, and Vite with full CI/CD DevSecOps practices.

### 🛠️ Features
- React + TypeScript frontend
- CI/CD with GitHub Actions
- Security scanning via Trivy
- GitOps deployment via ArgoCD
- Kubernetes-based container orchestration

### 🚀 Skills Used
React.js · TypeScript · GitHub Actions · ArgoCD · Trivy · Kubernetes · Vite
