### 🚗 Project Overview

Developed a full-stack parking management system initially in Java and later upgraded to Spring Boot to improve scalability and modularity.

### 🧩 Features
- User registration and authentication
- Car entry/exit tracking and billing
- Modular architecture using Spring Boot
- REST APIs with Spring MVC
- Database integration using JPA
- Secure access via Spring Security
- Unit testing with JUnit
- Embedded Tomcat for deployment

### 🚀 Skills Used
Java · Spring Boot · Spring Security · JPA · JDBC · JUnit · Tomcat
