### 🖼️ Project Overview

Designed and deployed a fully responsive photography portfolio website using AWS S3 and CloudFront, automated via Terraform Infrastructure as Code (IaC).

### 🛠️ Features
- Hosted static website on AWS S3
- Global delivery via CloudFront CDN
- Modular Terraform to manage infrastructure
- Interactive image carousel using HTML, CSS, JS (no libraries)
- Security via S3 bucket policies
- MIME type handling for optimized delivery
- CloudFront cache invalidation for instant content refresh
- Efficient versioning via Terraform remote state
