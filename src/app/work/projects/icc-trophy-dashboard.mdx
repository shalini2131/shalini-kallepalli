### 🏏 Project Overview

A Flask-based web app displaying India’s latest ICC match results with a visually rich UI and celebratory animations.

### 🛠️ Features
- Auto-fetches latest India match via cricket API
- Displays live scores and match details
- Confetti trigger when India wins
- Styled dashboard UI with background themes
- CI/CD pipeline using GitHub Actions
- Self-hosted GitHub runner on AWS EC2
- Dockerized & deployed on Kubernetes (kind)

### 🚀 Skills Used
Python · Flask · AWS EC2 · Docker · Kubernetes · GitHub Actions
